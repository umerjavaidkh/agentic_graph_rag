How ingestion works

  • Axis 1 (Document Structure): Always built first from the document parser.
  • Axis 2 (Semantic Enrichment): Automatically run when the server has an OpenAI key configured.
  • The result is exported as Neo4j import artifacts in output/ingestion/<job_id> and loaded into Neo4j automatically when configured.

How cypher ingestion works

  • The server executes your Cypher statements against the configured Neo4j.
  • Enable it with ALLOW_CYPHER_INGEST=true.
  • Use only in trusted environments (this can create/drop data).
